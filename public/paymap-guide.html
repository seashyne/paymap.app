PayMap v0.9.3 — คู่มือการใช้งานและการตั้งค่า

เวอร์ชันนี้เพิ่ม one-click demo โดยไม่ต้องมีบัญชี และรวมคู่มือการเปิดใช้งานทุกระบบหลัก

1) ทางเข้าเดโม

Personal

/demo/personal

Business

/demo/business

Merchant

/demo/merchant

Guide Page

/guide

2) ขั้นตอนติดตั้ง

npm install
cp .env.example .env.local
npx prisma generate
npx prisma db push
npm run db:seed
npm run dev

หลังจากนั้นเปิด http://localhost:3000/guide หรือเข้าเดโมผ่าน /demo/personal ได้ทันที

3) ตัวแปรแวดล้อมที่ต้องตั้งค่า

Database

DATABASE_URL, DIRECT_URL

Auth

NEXTAUTH_URL, NEXTAUTH_SECRET, AUTH_SECRET

Firebase

Client keys + FIREBASE_PROJECT_ID / CLIENT_EMAIL / PRIVATE_KEY

Stripe & Email

STRIPE_* และ RESEND_API_KEY

4) ระบบที่ใช้งานได้

Personal

Dashboard การเงินส่วนบุคคล, budget, savings, tax planner

Business

Employee management, payroll scaffold, leave requests, reports

Merchant

Inventory, suppliers, sales, VAT summary, top products

Billing / Settings

หน้า pricing, billing overview, profile/settings workspace

5) การตั้งค่าระบบจริง

หากต้องการ production จริง ควรตั้งค่าโดเมน, Firebase providers, Stripe webhook, อีเมลจาก Resend และสิทธิ์ฐานข้อมูลให้ครบ จากนั้นทดสอบ flow สมัครสมาชิก, login, checkout และ API namespaced ทุกโหมดอีกครั้ง

6) หมายเหตุ

เดโมแบบ one-click ใช้ผู้ใช้ตัวอย่างจาก prisma/seed.ts ดังนั้นถ้ายังไม่รัน seed หน้าเดโมจะไม่เข้า
