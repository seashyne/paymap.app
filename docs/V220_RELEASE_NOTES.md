# PayMap v22 Release Notes

## Highlights
- ปรับ Pricing model ใหม่ทั้ง Personal, Merchant และ Business
- เพิ่ม Free tiers สำหรับ Merchant และ Business
- ลดราคา Merchant ให้เหมาะกับร้านทั่วไปมากขึ้น
- อัปเดต Stripe pricing map ให้รองรับแผนใหม่
- แก้ bug ใน webhook และ checkout ที่ทำให้ price mapping ผิด
- ปรับหน้า pricing ให้ตรงกับ pricing model v22
- อัปเดต version เป็น 2.2.0
