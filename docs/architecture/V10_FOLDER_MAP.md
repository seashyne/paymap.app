# PayMap v10 Folder Map

- `src/app`: routes, layouts, route handlers
- `src/features/auth`: login/register/reset-password feature package
- `src/features/workspace`: workspace selection, switching, guards
- `src/features/planner`: planner domain
- `src/shared/components/ui`: canonical shared UI exports
- `src/shared/lib`: common helpers such as env, logger, errors, fetcher
- `src/server/db`: Prisma access and repositories
- `src/server/auth`: current-user, session, permissions
- `src/server/billing`: billing services and plans
- `src/server/storage`: file upload services
- `src/server/jobs`: queue and workers
