# PayMap v11.4.1 Public I18n + Overview Fix

Included in this pack:
- Landing page overview typography and spacing fixes
- Automatic public/auth language detection from browser Accept-Language
- English fallback when the browser language is not supported
- Public shell navigation/footer localization
- Login/register surface localization for English, Thai, and Lao
- Firebase auth provider locale aligned to the browser language

Notes:
- Language detection is based on browser/request language, not precise IP geolocation.
- Unsupported languages fall back to English.
