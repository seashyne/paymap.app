# PayMap v0.7 Release Notes

PayMap v0.7 expands the product from a multi-workspace finance dashboard into a more competitive finance platform.

## Highlights
- Personal finance intelligence: smart categorization suggestions, recurring detection, health score, spending insights, monthly summary.
- Business foundation: invoice/quotation scaffold, AR/AP preview, simple cashflow and P&L dashboard.
- Enterprise-ready foundation: organizations, teams, roles, approvals, audit, report-by-team, API/webhook scaffolding.
- Reliability roadmap documented: testing, monitoring, analytics, backup/restore, CI/CD, staging.

## What is production-ready vs scaffold
- Ready now: dashboard intelligence, workspace UX, release documentation, API placeholders, schema direction.
- Next implementation step: database migration, CRUD forms, RBAC enforcement, webhook delivery jobs, notification workers, SSO.
