# PayMap v5.2 Release Notes

## Highlights
- Dashboard Unification upgraded from v5.1 to v5.2 across Personal, Business, Merchant, and Enterprise.
- Added real module entry points for payroll, accounting, sales, inventory, enterprise controls, and executive reports.
- Added SaaS-grade Consent System with Legal Center, public Privacy page, enforced modal overlay, and audit visibility.
- Updated public/legal versioning to TOS v2.0.0 and Privacy v2.0.0.

## New Routes
- `/business/payroll`
- `/business/accounting`
- `/merchant/sales`
- `/merchant/inventory`
- `/merchant/accounting`
- `/enterprise/controls`
- `/enterprise/reports`
- `/settings/legal`

## Product Positioning
PayMap v5.2 focuses on wiring the dashboards to real operational flows instead of leaving modules behind API-only access.
