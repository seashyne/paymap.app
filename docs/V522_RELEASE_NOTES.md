# PayMap v5.2.2 — Edit / Approve Lifecycle Pass

## Highlights
- Dashboard workbenches upgraded from create-only hubs to lifecycle actions
- Business payroll now supports employee edit/archive, leave approve/reject, payroll approve/pay/reopen
- Merchant inventory now supports product edit/archive and sales status/void flow
- Enterprise controls now includes workspace member management actions inline
- Added multi-country Tax Workspace with detailed breakdown for TH, SG, MY, JP, US, GB, AU, DE and estimate mode for other countries
- Billing flow updated for clearer monthly/yearly handling in pricing and billing UI
- SaaS consent versions updated to TOS 2.1.0 / Privacy 2.1.0

## Notes
Tax engines in non-detailed countries remain estimator mode and should be verified locally before filing.
