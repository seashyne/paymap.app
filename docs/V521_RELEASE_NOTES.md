# PayMap v5.2.1 Release Notes

## Dashboard Action Wiring
- Added real create/update workbenches for Business Payroll, Business Accounting, Merchant Sales, Merchant Inventory, Merchant Accounting, and Enterprise Controls.
- Dashboard action panels now call production APIs directly instead of linking only to hub pages.
- Added inline live snapshots so operators can validate newly created records without leaving the dashboard.

## Business
- Create employee from dashboard
- Create leave request from dashboard
- Run payroll from dashboard
- Create chart of account and post manual journal from dashboard
- Load live trial balance from ledger engine

## Merchant
- Create product, supplier, and inventory adjustments from dashboard
- Create sales order from dashboard with stock-aware product selection
- Generate VAT report from dashboard
- Create merchant chart of accounts from dashboard

## Enterprise
- Create enterprise organizations from control center
- Create business/merchant workspaces from control center
- View live organization and workspace registry inline

## Version
- App version updated to 5.2.1
- Dashboard label updated to v5.2.1 Unified
- Release name updated to Deep Action Wiring
