# PayMap v11.4.3 Dashboard/App Localization Pack

This pack extends localization into logged-in surfaces with a desktop-first focus.

Included updates:
- App shell labels localized for en/th/lo
- Workspace switcher localized
- Dashboard shell copy localized
- Planner page and planner persistence surface localized
- Billing page shell and billing control center copy localized
- Settings page shell, tabs, and major controls localized
- Language fallback remains English

Notes:
- This pass focuses on the main logged-in surfaces customers use most on PC.
- Deep feature modules may still contain older product copy and can be localized incrementally in later passes.
