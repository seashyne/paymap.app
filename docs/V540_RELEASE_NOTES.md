# PayMap v5.4 Release Notes

## Headline
PayMap v5.4 introduces a production-ready Subscription Control System with a generous free tier, module-level feature gating, a new pricing experience, and deeper Stripe checkout wiring.

## What changed
- Added plan matrix and module feature-gates across personal, business, merchant, and enterprise modules
- Personal Free now includes wallets, budgets, savings, simulation, investments, installments, and financial reports
- Business Free includes payroll basic and invoices lite; accounting, tax, and reconciliation require Business SME+
- Merchant Free/Starter include sales and inventory basics; accounting, VAT, and reconciliation require Merchant Growth+
- Enterprise controls and executive reports now require Enterprise access
- Added upgraded pricing page and module lock UX
- Stripe checkout now accepts return paths for more precise billing flows
