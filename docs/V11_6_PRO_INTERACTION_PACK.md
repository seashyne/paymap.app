# PayMap v11.6 Pro Interaction Pack

This pack adds a stronger desktop interaction layer for PC-first workflows.

Included:
- Advanced table engine with sort, search, column visibility, and sticky headers
- Side panel system for row details
- Command actions integrated into the global search flow
- Expanded keyboard system (Ctrl+K, G then D/P/B/S/W, Esc closes panels, Ctrl/Cmd+Enter for submit)
- Live activity feed for realtime dashboard surfaces
- Optimistic transaction updates
- Density modes: comfortable, compact, dense

- `src/components/ui/AdvancedTable.tsx`
- `src/components/ui/SidePanel.tsx`
- `src/components/dashboard/TransactionTable.tsx`
- `src/components/dashboard/ActivityFeed.tsx`
- `src/components/dashboard/CommandActionCenter.tsx`
- `src/components/ui/GlobalSearch.tsx`
- `src/components/layout/DesktopUXControls.tsx`
