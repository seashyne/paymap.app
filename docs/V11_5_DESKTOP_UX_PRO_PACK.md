# PayMap v11.5 Desktop UX Pro Pack

This pack adds a desktop-first UX layer to the workspace experience.

## Added
- Linear-style sidebar polish
- Dense mode toggle
- keyboard navigation shortcuts
- command palette polish (`Ctrl+K`)
- quick action rail on dashboard
- table-first recent transactions section
- realtime dashboard panel using SSE stream
