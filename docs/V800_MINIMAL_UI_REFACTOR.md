# PayMap v8 Minimal UI Refactor

## สิ่งที่เปลี่ยน
- ปรับเวอร์ชันเป็น 8.0.0
- ปรับสีและพื้นหลังให้มินิมอล สะอาดตา ลด glow และ gradient ที่ดูหนักเกินไป
- ลบข้อความเชิง developer / setup ที่ลูกค้าไม่จำเป็นต้องเห็น
- ปรับหน้า Home, Login, Register, Download, Guide ให้สื่อสารกับลูกค้ามากขึ้น
- เปลี่ยน Guide จากคู่มือติดตั้งระบบ เป็นหน้าเริ่มต้นใช้งานสำหรับลูกค้า

## เป้าหมาย
- ให้ PayMap ดูเป็น product ที่เรียบขึ้น อ่านง่ายขึ้น
- เน้นประสบการณ์ลูกค้า มากกว่าการอธิบายระบบหลังบ้าน
